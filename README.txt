* 操作方法
  
  キーボード
     W : 前進
     S : 後退
     A : 左を向く
     D : 右を向く
     J : 岩を投げる
     K : ジャンプ
     V : スクリーンショットをとる
     N : ポーズ

     Esc : プログラム終了
